GDB Debugger

  1. (gdb)file name
    load executable file "name" into the GDB debugger;
  2. (gdb)l number
    list source codes, which is loaded from executalbe file "name", from line "number"

VI/VIM Usage

  1. How to replace specific string within a file?

    :%s/old_name/new_name/g
    Replace old_name with new_name, 'g' is a global parameter, that is , you will replace all the old_names within the file;

  2. How to copy x lines from file A to file B

    In file A, change the mode via ESC key, input "xyy+"(that means VIM will copy x lines from the line where the cursor exists and store them in the register). Then you could exit file A and open file B, move the cursor to the position where you want to paste -- change the mode via ESC key and press key p, then they would be pasted in file B
